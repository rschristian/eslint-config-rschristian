# eslint-config-rschristian

My personal ESLint config. Includes Preact, React-Hooks, Prettier, and TS support.

## Install

```
$ yarn add eslint eslint-config-rschristian
```

## Usage

In your `.eslintrc.js` file:

```js
module.exports = {
    extends: 'rschristian',
    rules: {
        ...
    }
}
```
